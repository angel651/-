### 零、技术选型

- 基础框架 SpringBoot  

- 通信协议 WebSocket  

- 数据存储 MySQL  

### 一、项目功能

- 用户模块：用户注册、用户登录
- 好友模块：添加好友、删除好友、好友申请及处理
- 消息模块：获取消息记录
